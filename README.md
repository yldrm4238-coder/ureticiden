# Üreticiden - Çiftçi Bağlantı Platformu

Çiftçilerin taze ürünlerini doğrudan alıcılara ulaştırabileceği tarım pazaryeri platformu.

## Kurulum ve Çalıştırma

Projeyi yerel ortamınızda çalıştırmak için:

```bash
# Bağımlılıkları yükleyin
bun install

# Geliştirme sunucusunu başlatın
bun dev
```

## Özellikler
- **Üretici Paneli:** Çiftçiler kolayca ürünlerini sisteme ekleyebilir.
- **Pazar Yeri:** Alıcılar kategorilere veya şehirlere göre taze ürün arayabilir.
- **Canlı Veri:** Sistem doğrudan Supabase veritabanıyla entegredir.
